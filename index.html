Sleep Stages Classification of EEG Signals

Objective

To develop an automated system that accurately classifies EEG signals into different sleep stages using machine learning and deep learning models.

Approach

  • Used the Sleep-EDF dataset containing EEG recordings totaling 23.6 hours.
  • Segmented EEG signals into 30-second epochs for analysis.
  • Extracted time-frequency features using Pseudo Wigner-Ville Distribution.
  • Converted feature matrices into images for input to ML/DL classifiers.
  • Tested multiple classifiers including Logistic Regression, KNN, Decision Trees, Random Forest, and MLP.

Results

MLP model achieved highest performance with 93.1% accuracy in classifying 5 sleep stages.

Object Detection in Dark Images

Objective

To improve object detection accuracy in dark and low‑visibility environments by combining advanced image enhancement with a modern deep learning detector.

Approach

  • Use the ExDark dataset containing 7,363 low‑light images with annotations for 12 object classes.
  • Enhance image quality using MIRNet to improve visibility and reduce noise prior to detection.
  • Train and evaluate YOLO v8n on the enhanced dataset with normalized annotations.
  • Preprocess images (resize, RGB conversion, normalization) and fine‑tune the model with GPU acceleration.

Results

  • Achieved 70.2% mAP50, demonstrating improved detection in dark imagery.
  • Best‑performing class (“Bus”) reached 90.4% mAP50 in evaluations.
